Slicer 4.0

Easy to handle slicing software

Prepare your stl. file to be 3D printable with 3DGence Slicer 4.0, that gives you the wide choice of the pre-defined settings and useful functionality. It allows you for basic modification of models (rotating, scaling, mergint etc.), but also for changing all advanced print options. The key feature is the prepared material profiles by R&D department, that make you work easier and faster.

Cloud

Online platform for 3D printing management

Monitor your 3D printer through the network with 3DGence Cloud, that enables remote communication with machine. It allows you to fully control the 3D printing process, including runing and queuing printouts, collection of usage statistics and service functions. This solution definitely enhance your workflow, all you need is internet access.
